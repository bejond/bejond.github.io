markdown复选框为: 
```
* [ ] 这是未选中的复选框格式
* [x] 这是选中的复选框格式
```
* [ ] 这是未选中的复选框格式
* [x] 这是选中的复选框格式

部分编辑器好像没有复选框效果，无所谓了，自己看明白就好:joy:。
